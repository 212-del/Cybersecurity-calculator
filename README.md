# 🔐 Cybersecurity Calculator

Cybersecurity Calculator is a lightweight toolkit designed to perform common security-related calculations and utilities used in cybersecurity learning and practice.

This project was built as a hands-on way to understand core cybersecurity concepts such as cryptography, password security, hashing, and encoding.

## 🚀 Features

- Password strength checker
- Password entropy calculator
- Hash generator (MD5, SHA-1, SHA-256)
- Encoding/Decoding tools (Base64, etc.)
- Simple cryptographic utilities
- Beginner-friendly CLI interface

## 🎯 Purpose

This project is part of my journey into cybersecurity. It demonstrates practical understanding of security fundamentals and Python programming.

## 🛠️ Tech Stack

- Cryptography libraries
- CLI-based interface
